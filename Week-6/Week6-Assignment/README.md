Week 6: Mobile Application Networking, API & JSON Tasks

Overview
This repository contains Week 6 class tasks for Mobile Application Development. The tasks focus on networking concepts, API investigation, and JSON data structures used in mobile applications.

---

 Project Structure

- Task 1: Mobile Application Networking Analysis
- Task 2: API Investigation
- Task 3: JSON Creation



Task 1: Mobile Application Networking Analysis

This task identifies client-server architecture and data exchange in popular mobile applications such as WhatsApp, Facebook, Uber, M-Pesa, and YouTube.

File: `Task1-Networking-Analysis/networking_analysis.md`



Task 2: API Investigation

This task explores a public API (OpenWeatherMap API) and explains its purpose, GET request format, and expected JSON response.

File: `Task2-API-Investigation/api_investigation.md`


Task 3: JSON Creation

This task demonstrates how structured data is represented using JSON format for student information, including single and multiple records.

File: `Task3-JSON-Creation/json_creation.json`


Technologies Used

- JSON (Data Format)
- HTTP/REST API Concepts
- Mobile Application Networking Concepts



 Learning Outcomes

- Understanding client-server architecture in mobile apps
- Understanding how APIs work using HTTP GET requests
- Creating and structuring JSON data
- Applying networking concepts in real-world applications
