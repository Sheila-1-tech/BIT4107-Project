Week 3: User Interface (UI) Development
Weekly Objectives
Understand the complete UI/UX practical workflow.
Build a complete UI prototype consisting of at least five screens.
Activities Completed
Designed initial wireframes and graphical user interface layouts for the application screens using Figma.
Established responsive design principles, including appropriate selection of color schemes and typography.
Converted Figma designs into functional cross-platform Flutter layouts.
Implemented user interaction features, including form handling, validation logic, and screen transitions.
Conducted emulator testing to evaluate mobile responsiveness, spacing, and overall usability across different screen sizes.
Challenges & Solutions
Challenge: Ensuring that form input fields remained responsive and well-aligned across different screen sizes.
Solution: This was addressed by applying flexible layout widgets and structured padding to maintain consistent alignment and visual balance.
