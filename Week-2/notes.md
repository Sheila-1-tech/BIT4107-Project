Week 2: Programming Languages and Frameworks
Weekly Objectives

The objective of this week was to strengthen understanding of core programming syntax while applying it to the development of practical user interface components. In addition, the week focused on designing and implementing a Mini Student Management Application.

Activities Completed

During this week, I completed the following tasks:

Developed fundamental UI components, including buttons, text input fields, and navigation bars.
Designed and implemented a login interface and a student registration screen as part of the application.
Applied basic navigation techniques to enable movement between different screens within the app.
Implemented simple local data handling to simulate record storage functionality.
Initially used Android Studio for development before transitioning the project setup to Flutter for further enhancement.

Challenges & Solutions
Challenge: Difficulty was encountered when linking UI components to functional navigation between screens.
Solution: This was resolved by reviewing lecture materials and practicing implementation of cross-screen navigation techniques until the desired functionality was achieved.
