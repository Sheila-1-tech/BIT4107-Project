# Task 2: Student Database Design

Database Name: StudentManagementDB

 Table: Students

| Field Name   | Data Type | Description |
|--------------|----------|-------------|
| student_id   | INT      | Primary Key (Unique Identifier) |
| name         | VARCHAR  | Student full name |
| course       | VARCHAR  | Course enrolled |
| year         | INT      | Year of study |
| phone_number | VARCHAR  | Contact number |

---

Primary Key:
- student_id
